# 📱 微信客服功能配置指南

## 🎯 开通客服功能

### 步骤1: 登录微信公众平台

1. 访问 https://mp.weixin.qq.com/
2. 使用小程序管理员账号登录
3. 选择你的小程序

### 步骤2: 开通客服功能

```
左侧菜单 → 功能 → 客服功能
    ↓
点击"开通"按钮
    ↓
同意服务条款
    ↓
开通成功 ✅
```

### 步骤3: 添加客服人员

1. 进入"客服功能"管理页面
2. 点击"添加客服"
3. 输入客服微信号
4. 发送邀请
5. 客服接受邀请 ✅

**客服人数限制**:
- 普通小程序: 最多添加100个客服
- 认证小程序: 无限制

### 2. 设置自动回复

**常见问题快捷回复**:

#### Q1: 如何使用？
```
📖 使用教程：

1️⃣ 复制视频链接
   - 打开抖音/快手/小红书等APP
   - 找到要去水印的视频
   - 点击分享 → 复制链接

2️⃣ 粘贴到小程序
   - 打开PureClip小程序
   - 粘贴链接到输入框
   - 点击"一键去水印"

3️⃣ 获取无水印视频
   - 等待2-5秒完成解析
   - 点击"播放"或"下载"
   - 保存到相册 ✅

支持平台：抖音、快手、小红书、百度等
```

#### Q2: 视频解析失败？
```
🔧 解决方案：

1. 检查链接是否正确
   - 确保复制了完整的分享链接
   - 包含 https:// 开头

2. 检查网络连接
   - 确保网络稳定
   - 尝试切换WiFi/4G

3. 重新尝试
   - 返回首页重新提交
   - 或等待1分钟后再试

如仍有问题，请发送：
• 视频平台（如：抖音）
• 错误提示（截图）
• 视频链接
```

#### Q3: 下载失败？
```
💾 下载问题解决：

1. 检查相册权限
   - 打开微信设置
   - 隐私 → 照片
   - 允许访问相册

2. 检查存储空间
   - 确保手机有足够空间
   - 至少预留500MB

3. 尝试复制链接
   - 点击"复制链接"
   - 在浏览器中打开下载

需要帮助？发送"人工客服"
```

## 🔔 消息通知设置

### 客服APP通知

**设置路径**: 微信客服APP → 我的 → 通知设置

**推荐设置**:
- ✅ 新消息提醒: 开启
- ✅ 声音: 开启
- ✅ 震动: 开启
- ✅ 横幅显示: 开启

### 客服在线状态

**在线**: 
- 客服登录APP/网页版即为在线
- 用户可以立即收到回复

**离线**: 
- 客服未登录或APP在后台
- 消息会暂存，登录后可查看
- 建议设置自动回复："客服离线，留言会在上线后回复"

## 🚫 注意事项

### 1. 48小时限制

- ⚠️ 只能在用户最后一次发消息后48小时内回复
- ⚠️ 超过48小时无法主动发送消息
- ✅ 用户再次发送消息后，时间窗口重新计算

### 2. 消息频率

- ⚠️ 每个用户每天最多接收100条客服消息
- ✅ 合理控制回复频率
- ✅ 避免发送过多营销信息

### 3. 违规内容

**禁止发送**:
- ❌ 违法违规内容
- ❌ 色情暴力信息
- ❌ 诱导分享/关注
- ❌ 虚假宣传

**违规后果**:
- 客服功能被限制
- 小程序被封禁

## ❓ 常见问题

### Q1: 点击"联系客服"提示"商家暂未提供客服服务"

**A**: 
1. 检查是否已在微信公众平台开通客服功能
2. 检查是否已添加客服人员
3. 检查客服人员是否已接受邀请

### Q2: 客服收不到用户消息

**A**:
1. 确认客服已登录APP或网页版
2. 检查小程序是否选择正确
3. 尝试退出重新登录

### Q3: 用户收不到客服回复

**A**:
1. 检查是否在48小时窗口内
2. 检查用户是否拉黑了小程序
3. 检查网络连接是否正常

### Q4: 如何添加多个客服？

**A**:
1. 进入微信公众平台
2. 客服功能 → 添加客服
3. 输入客服微信号
4. 发送邀请链接
5. 客服点击接受
